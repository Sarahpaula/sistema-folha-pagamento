# Sistema de Folha de Pagamento - Gestão de Colaboradores

Projeto desenvolvido para consolidar fundamentos de **Programação Orientada a Objetos (POO)** em Java.

## 🚀 Tecnologias e Conceitos Aplicados
- **Linguagem:** Java (JDK 17+)
- **Herança & Polimorfismo:** Uso de classe abstrata `Funcionario` e subclasses especializadas.
- **Encapsulamento:** Atributos protegidos e métodos de acesso.
- **Estruturas de Dados:** Gerenciamento dinâmico com `ArrayList`.
- **Tratamento de Dados:** Configuração de `Locale` para suporte a diferentes padrões decimais.

## 🛠️ Como rodar o projeto
1. Clone o repositório.
2. Abra em uma IDE (IntelliJ, VS Code ou Eclipse).
3. Execute a classe `Main.java`.

## 📝 Regras de Negócio
O sistema calcula o salário final com base em três modalidades:
1. **Padrão:** Salário base fixo.
2. **Comissionado:** Salário base + % sobre vendas.
3. **Produção:** Salário base + bônus por peça produzida.
